February 15, 2017

This article will continue the topic of Windows command & hotkeys. Part-1 includes all hotkeys and short command lines for Run windnow dialog. The rest of this topic will focus the most useful commands, how to create a batch script.

Most common and useful commands

help

There is a common mistake for beginners, or even some developers. They forget to check help command before they Google or try it, when they hit some impediment or roadblock. Actually help command is most common feature in any software or tool. Learn how to use help command is first important than any other commands.

  • Start command prompt and type help. You will get a list of command which you can use, and short decription of each command.
  • Use help command to learn other commands C:\>help C:\>help cd
