     Ubuntu 18 & Kubernetes

    Prerequisites You are familiar with Ubuntu, at least you have some experience working on Linux system. You are familiar with basic bash/shell command Install Hypervisor Choose Virtualbox or KVM (I chose KVM) Check that your CPUeEEEEEE supports hardware virtualization egrep -c '(vmx|svm)' /proc/cpuinfo If 0 it means that your CPU doesn’t support hardware virtualization. If 1 or more it does - but you still need to make sure that virtualization is enabled in the BIOS.
